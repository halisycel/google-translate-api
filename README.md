# google-translate-api
```
py scraper.py --sl-{source language} --tl-{target language} {text}

py scraper.py {text} (In this state, it translates from English to Turkish.)
```

```
IN:

C:\Users\halisycel\google-translate-api> py scraper.py --sl-tr --tl-en hey adamım, bu uygulama çalışıyor.

OUT:

{'main_result': 'hey man, this app works. '}
```
